# spider_legal
爬取法律法规爬虫

**特殊说明**:
| Python库 | 用途 | 安装 | 备注 |
|:-------|:-------------|:----------|:----------|
| scrapy | 爬虫框架 | *pip3 install scrapy* | |
| PyExecJS  | 执行js | *pip3 install PyExecJS* |  |
| redis  | 生成或获取redis队列 | *pip3 install redis* |  |
| pymysql  | mysql数据库驱动 | *pip3 install pymysql* |  |
